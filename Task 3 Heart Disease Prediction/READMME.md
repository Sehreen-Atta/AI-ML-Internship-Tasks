# ❤️ Heart Disease Prediction

## 🎯 Objective
To predict whether a person has heart disease using machine learning.

## 📊 Dataset
A heart disease dataset containing medical attributes such as age, sex, chest pain type, and cholesterol.

## ⚙️ Steps Performed
- Loaded dataset from local file
- Selected 5 important features
- Converted categorical data into numeric format
- Split data into training and testing sets
- Applied Logistic Regression model
- Evaluated using:
  - Accuracy
  - Confusion Matrix
  - ROC Curve

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 📌 Conclusion
The model successfully predicts heart disease with good accuracy using a simplified dataset.

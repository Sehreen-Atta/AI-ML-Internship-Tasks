# 📈 Stock Price Prediction

## 🎯 Objective
Predict the next day's stock closing price using machine learning.

---

## 📊 Work Done
- Downloaded stock data using yfinance
- Prepared dataset for prediction
- Applied Linear Regression model
- Predicted next-day closing prices
- Compared actual vs predicted values

---

## 🛠️ Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- yfinance

---

## 📌 Result
- Successfully trained model
- Visualized predictions
- Evaluated performance using R² score

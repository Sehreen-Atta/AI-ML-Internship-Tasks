# 🌸 Iris Dataset Visualization

## 🎯 Objective
To explore and visualize the Iris dataset to understand relationships and distributions.

---

## 📊 Work Done
- Loaded dataset using seaborn
- Displayed first rows using `.head()`
- Checked shape and columns
- Used `.info()` and `.describe()`
- Created:
  - Scatter Plot
  - Histogram
  - Box Plot
- Derived insights from data

---

## 🛠️ Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

---

## 💡 Insights
- Dataset contains 150 rows and 5 columns
- No missing values found
- Petal length clearly separates species
- Setosa species is easily identifiable
